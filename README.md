# yesno.wtf
Wrapper for [yesno.wtf](https://yesno.wtf/)

# Install
Python 3 or higher is required.    
```cmd
# Linux/OS X
$ python -m pip install -U yesno.wtf

# Windows
> py -3 -m pip install -U yesno.wtf
```    

# Example 
```py
from yesno import YesNo

yesno = YesNo()

print(yesno.answer)
```

# Method
- `answer`
- `forced`
- `image`
